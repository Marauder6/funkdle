# Project Fallout Junk-dle -> Funkdle!

A [Wordle]([url](https://www.nytimes.com/games/wordle/index.html))-inspired guessing game based on Fallout 4 junk item properties.
Work to guess the target item, recieving feedback on the following categories:
- Weight
- Value
- Component types
- Number of overall components
- Number of unique component types
- DLC area
